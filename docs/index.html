About

The LXON serialization format is a better JSON alternative developed solely by Nicholas (Jasper) Birla-Eliade.

It introduces twice the container types and quadruple the value types. It also uses far less syntax, making it much smaller (often up to half a JSON file) and far easier to read, write and edit.
