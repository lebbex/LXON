Outline

Values are structureless independant chunks of data. They're the entries of stored data.

Containers are also values and can be used anywhere a value is expected, although "value" often refers to the non-container values, such as on this page and all it's sub-pages in majority.

Static Values

Static value are values that don't actually take in data. All they do is define a single preset value using a declaration character.

Here is a cheat sheet list of all static values and their respective declaration characters:

  • True$
  • False!
  • Null~
  • Undefined_
  • NaN+ -*Must be followed by a non-numerical character
  • Positive (+) Infinity% +%
  • Negative (-) Infinity-%

Dynamic Values

Dynamic values are values that take in data. They use a value type declaration symbol at the start, or are self defined in certain cases, and then expect additional data, which will be the acctual information stored itself. The type declaration only announces how the value should be interpreted.

Here is a cheat sheet list of all dynamic values and their respective declaration characters:

  • Full String"
  • String`
  • Char'
  • Number
  • Date&
  • Keybind@
  • SRGB Color#
  • Linear Color (8bit)=
  • Linear Color (16bit)/
  • Enum*
  • Binary?
