Definition

In LXON, "text" is a way to group all string-like types into one. Text values encapsulate three distinct yet similar values types:

  • Full String"
  • Regular String`
  • String Character'

Text values are values that get turned into string by default. Certain languages support a dinstinct single-character string value type (char), however most don't, which is why the default behavior is for everything to be simplified to string.

When serializing the data, the string automatically choses the correct text type based on whether it's the last value on the line, and, in the case of a language that doesn't support char, if the length of the string is of 1.
