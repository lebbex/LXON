Regular Keys

Regular keys are used by !! Containers / Object ! objects !. They can only be text with a specific format and syntax. Many characters aren't allowed, such as starting whitespace and colons. Escape characters (\n, \s, etc.) also aren't supported.

Usage

To use a regular key, simply create a !! Containers / Object ! object ! and write your key as is. Here's some example usage as reference: 

{
key1: "string

key2:{
	a: 10
	b: 20

key3:
	3key: "value

Typed Keys

Typed keys are used by !! Containers / Map ! maps ! and support keys of text and non-text types. Also, string keys are superior to regular text keys since they support escape characters, allowing starting whitespace and colons. All typed keys of a map must be of the same type, hence typed keys are homogeniously typed.

Here is a cheat sheet list of all typed key types and their respective declaration characters:

  • String" `
  • Char'
  • Boolean
  • Number
  • Date&
  • Keybind@
  • SRGB Color#
  • Linear Color= /
  • Enum*

Usage

To use a key, simply create a !! Containers / Map ! map ! and declare the key type using a respective declaration character (see above).

Here's some example usage of various values as inspiration: 

("
key\n1: "string

key2:('
	a: 10
	b: 20

key3:(
	3.2: "value
	-.4: "value

key4:(&
	2026-07: "value
	2026-07-01T12:30:: "value

key5:(*weapon
	sword: 42
	gun: 83
	hammer: 0
